To compile the program:

	1) cd to this folder.
	2) Open command or terminal.
	3) Compile all java classes with javac.


		javac Sort.java Insertionsort.java Mergesort.java Quicksort.java


	4) Run program in cmd or terminal.

		java Sort

	5) Program instructions are displayed on cmd/terminal.
